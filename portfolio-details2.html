" 롯데월드를 벤치마킹하여 구현한
놀이공원 홈페이지입니다. "

기획배경

  • 사용자의 선호도와 취향을 파악하여 여러가지 서비스를 추천
  • 놀이기구 예약을 통한 사용자의 편의성 증가
  • 놀이공원에 방문하지 않아도 기념품 구매가 가능한 서비스

기능구현

  • 파크 이용안내, 오시는 길, 가이드맵 지도 API
  • 편의시설 위치보기, 지도 API
  • 어트랙션, 기프트샵, 식당, 영화관 운휴 CRUD
  • 카테고리 및 해시태그 관리

페이지 상세설명


이용가이드의 '파크이용안내' 페이지입니다.

파크 이용안내에 들어가시면 놀이공원을 이용 시에
주의해야 할 사항들이 적혀 있습니다.
이용가이드의 '편의시설' 페이지입니다.

편의시설 페이지는 편의시설의 간단한 정보와
위치들을 보실 수 있습니다.
이용가이드의 '가이드맵' 페이지입니다.

가이드맵 페이지는 편의시설과 어트랙션의 각 해당 위치를 확인하실 수 있습니다.
관리자 파트의 '카테고리 등록' 페이지입니다.

식당의 카테고리를 설정하여 등록할 수 있습니다.
관리자 파트의 '카테고리 수정' 페이지입니다.

등록된 식당의 카테고리 중 선택하여 수정할 수 있습니다.
관리자 파트의 '카테고리 삭제' 페이지입니다.

원하는 식당의 카테고리를 선택하여 삭제할 수 있습니다.
관리자 파트의 '해시태그 등록' 페이지입니다.

어트랙션, 영화관, 페스티벌, 기프트샵 등에 설정하는
해시태그를 등록하는 페이지입니다.
관리자 파트의 '해시태그 삭제' 페이지입니다.

등록되어 있는 태그를 선택하여 삭제할 수 있습니다.
관리자 페이지의 '운휴 등록' 페이지입니다.

운휴 일정 등록 버튼을 누르시면, 운휴를 등록하고 싶은
어트랙션을 선택 후 날짜를 선택하여 등록하실 수 있습니다.

프로젝트 후기

  • Ajax를 사용하는 것과 Open API를 처음 적용하는 데에 있어 어려움이 있었지만 실제로 프로젝트에 적용시킴으로써
    좀 더 이해할 수 있었습니다.
  • 시간이 부족하여 운휴일정과 관련된 날짜에 대해서 유효성 검사를 좀 더 확실하게 하지 못한 것이 아쉬웠습니다.
  • 사용자의 정적 페이지를 다소 많이 맡은 것 같아 좀 아쉬움이 남아있었지만 그래도 관리자의 CRUD를 도맡아 해서
    구현에는 어려웠지만 뿌듯하고 재미있었습니다.
  • 팀원들과 많은 회의와 소통을 통해 협업하면서, 문제를 해결하고, 다양한 기능을 구현하면서 페이지의 완성도를 높일수록
    뿌듯했습니다.
