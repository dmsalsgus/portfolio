Project information

  • Project Name : Daitso
  • Category : Java
  • Language : Java
  • Tool : Eclipse, Sourcetree, Git, Google Drive
  • Project date : 2023.08.16 ~ 2023.08.24
  • Github : Daitso

구현목표

  • 중고 거래 플랫폼을 이용하는 사용자들이 신뢰도 있는
    거래를 위한 우편함을 구현한다.
  • 알림 키워드에 대한 직접적인 검색이 가능하도록 구현한다.

기능구현

  • 물품 구매 : 물품선택, 물품검색, 물품정렬
    (가격순, 날짜별, 인기별, 나눔)
  • 카테고리를 선택 후, 상품의 키워드 검색
  • 원하는 알림 키워드를 설정하는 기능
  • 더미 데이터 생성
