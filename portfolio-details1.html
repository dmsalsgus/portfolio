" 롯데월드를 벤치마킹하여 구현한
놀이공원 홈페이지입니다. "

기획배경

  • 사용자의 선호도와 취향을 파악하여 여러가지 서비스를 추천
  • 놀이기구 예약을 통한 사용자의 편의성 증가
  • 놀이공원에 방문하지 않아도 기념품 구매가 가능한 서비스

기능구현

  • 파크 이용안내, 오시는 길, 가이드맵 지도 API
  • 편의시설 목록보기, 상세보기, 지도 API, 편의시설 CRUD
  • 오픈 채팅 기능
  • 어트랙션, 기프트샵, 식당, 영화관 운휴 CRUD

페이지 상세설명


이용가이드의 '파크이용안내' 페이지입니다.

파크 이용안내에 들어가시면 놀이공원을 이용 시에
주의해야 할 사항들이 적혀 있습니다.
이용가이드의 '편의시설' 페이지입니다.
편의시설 페이지에 들어가시면 놀이공원에서 이용하실 수 있는
모든 편의시설을 볼 수 있는 목록들을 보실 수 있습니다.
이용가이드의 '편의시설 상세' 페이지입니다.

편의시설 상세 페이지는 편의시설의 간단한 정보와
운영시간 및 전화번호를 확인하실 수 있습니다.
이용가이드의 '편의시설 상세' 페이지입니다.
또한 편의시설이 현재 가지고 있는 위치의 정보를 카카오맵 지도 API를 통해
해당 위치를 확인하실 수 있습니다.
이용가이드의 '가이드맵' 페이지입니다.
가이드맵의 페이지는 편의시설의 위치뿐만 아니라 위치 정보를 가지고 있는
7개의 시설(어트랙션, 편의시설, 영화관, 페스티벌, 포토존, 식당, 기프트샵)의
전체 지도와 같이 위치를 확인하실 수 있습니다.
이용가이드의 '가이드맵' 페이지입니다.
현재 존재하는 시설들의 이름 정보가 출력되며, 해당 시설을 선택시 선택하신 시설의 위치 정보를 유효하게 표현할 수 있습니다.
'OpenChat' 페이지입니다.
닉네임을 입력하고 들어가기 버튼을 누르면 해당 닉네임으로 오픈 채팅방에
입장하여 다른 사람들과 놀이공원의 여러 정보를 공유를 할 수 있습니다.
Web socket을 OpenChat에 적용하여 실시간으로 양방향 통신이 가능합니다.
관리자 페이지의 '운휴' 페이지입니다.
운휴가 등록되어 있는 어트랙션의 정보를 확인하실 수 있습니다.
관리자 페이지의 '운휴' 페이지입니다.
운휴 일정 등록 버튼을 누르시면, 운휴를 등록하고 싶은 어트랙션을 선택 후
현재 날짜 이후로 날짜를 선택하여 등록하실 수 있습니다.
관리자 페이지의 '운휴' 페이지입니다.
등록된 어트랙션에 한에 원하는 어트랙션을 선택하신 후, 운휴일정을 삭제하실 수 있습니다.

프로젝트 후기

kakao map API 파트를 담당하였는데, 실제 지도 위에 DB에 저장한 위치를 나타내고 마커를 다양하게 사용해본 것이 재미있었습니다.
또한 Socket을 구현함으로써 server에 대한 이해도를 높일 수 있었습니다. 팀원들과의 소통을 통해 소스 코드나 css공유 등 페이지의 통일성을 주었고,
문제가 발생하면 회의를 통해 수월하게 해결할 수 있었습니다. 프로젝트를 직접 기획부터 구현까지 완성함으로써 굉장히 만족도가 높았던 프로젝트지만
한편으로는 좀 더 유효성을 확실하게 처리했으면 하는 아쉬움도 남는 프로젝트였습니다.
이번 프로젝트를 통해 협업이 얼마나 중요한지에 대해 배울 수 있었던 좋은 경험이었습니다.
